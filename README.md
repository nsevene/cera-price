## 🎬 Часть 2. Производство ИИ-видеоконтента

**Основная задача:** Создание **8 длинных презентационных видео** (по одному ролику для каждой из 8 секций главного каталога) с "оживлением" фотографий через генеративную нейросеть **Hailuo AI Video**, а также **динамичный перемонтаж стокового видео шоурума**. 

### 🔄 Как строится процесс работы над видео:
1. 🧠 **Промптинг и ИИ Генерация (8 секций):** Я вручную обрабатываю исходные фотографии для каждой секции. Задаю лучшие текстовые промпты, отбраковываю неудачные дубли с артефактами и перегенерирую сцены до достижения идеального кинематографичного качества через алгоритмы Hailuo.
2. 🏙️ **Перемонтаж видео Шоурума:** Глубокая работа со стоковыми исходниками живого шоурума: отбор лучших пролетов камеры, цветокоррекция под общую стилистику бренда, адаптация ритмики монтажа.
3. 🎞️ **Post-Production (Монтаж):** Бесшовная склейка, профессиональный саунд-дизайн (наложение музыки, интро-звуковых эффектов) и добавление брендированных титров и инфографики.
4. 📱 **Экспорт (Multi-Format):** Каждое из 9 видео (8 секций + шоурум) рендерится сразу в **два формата** для максимального охвата:
   * **Desktop.** Горизонтальный широкоформатный (16:9).
   * **Mobile.** Вертикальный/Квадратный формат (9:16 / 4:5) с ручным и умным кадрированием центра внимания для мобильной версии сайта.

### 💰 Финансовая смета (ИИ Производство и Монтаж)
* **Анимация и монтаж длинных ИИ-роликов (8 секций в 2-х форматах)**: **$1,600**
* **Профессиональный перемонтаж видео шоурума (в 2-х форматах)**: **$200**
* **Итоговая стоимость всего видеопроизводства:** **$1,800**
* ⏳ **Сроки реализации:** 3–4 недели работы.
