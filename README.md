## Часть 1. Оценка стоимости реализованного кода (Тема CERA + Плагины)

Текущая платформа представляет собой сложное, высокотехнологичное решение (уровень Senior/Awwwards), базирующееся на WordPress, с использованием ReactJS, GSAP и нативных 3D Canvas технологий.

Общий объем работ по готовому коду оценивается примерно в **280 – 380 часов** высококвалифицированной разработки.

### 1.1 Детализация сметы по компонентам кода

**1. Фронтенд: WebGL / 3D Canvas (Плагин Cera 3D Catalog)**
*Самая технически сложная часть проекта.*
- Процедурная генерация текстур мрамора (Canvas API).
- Кинематографичный многоступенчатый прелоадер (GSAP, SVG-анимации, гравитация частиц).
- Уникальный физический движок скролла (инерция, параллакс, 3D-перспектива).
- Динамическая фильтрация WooCommerce без перезагрузок.
- **Стоимость (со скидкой 30%):** **$1,750 – $2,800** *(вместо $2,500 – $4,000)*

**2. Frontend: ReactJS Слайдер (Плагин Cera Portfolio Elementor)**
- Настройка Webpack/Babel сборки внутри WP.
- Три режима (Stream, Gallery, 3D Stack для мобильных).
- Связка с Elementor API.
- **Стоимость (со скидкой 30%):** **$1,400 – $2,100** *(вместо $2,000 – $3,000)*

**3. Backend: Архитектура темы CERA и CERA Helper**
- Разработка Custom Post Types и таксономий.
- Создание 21 уникального виджета для Elementor.
- Сквозная логика "пустышек" (состояний без контента).
- **Стоимость (со скидкой 30%):** **$1,050 – $1,750** *(вместо $1,500 – $2,500)*

**4. Верстка и UI/UX Анимации (Тема CERA)**
- Анимации доскролла (Ambient effects).
- Логика автопереходов с прогресс-баром и таймером прерывания.
- "Орбы" (плавающие шары) и адаптивность.
- **Стоимость (со скидкой 30%):** **$700 – $1,260** *(вместо $1,000 – $1,800)*

**5. Интеграция WooCommerce (Премиум-кастомизация)**
- Переопределение стандартов Woo для создания полноэкранных карточек товара ("чистый" дизайн).
- Безопасная AJAX-форма "Запросить образец".
- **Стоимость (со скидкой 30%):** **$560 – $840** *(вместо $800 – $1,200)*

### 💰 Итоговая оценка реализованного кода (Со скидкой 30%):
> **Рекомендованная цена для заказчика:** **$6,650 – $7,700**
*(Нижний порог рынка фриланса: ~$5,460 | Уровень премиальной студии: ~$8,750)*

## Часть 2. Смета на производство ИИ-видеоконтента (Hailuo AI)

**Задача:** Создание 42 премиальных видеороликов (по 2-3 минуты), где каждый ролик собирается из ~14 фотографий, "оживленных" через генеративную нейросеть **Hailuo AI Video**.

### Описание процесса (Workflow)
1. **Подготовка и ИИ Генерация:** Обработка 588 фото (42 видео × ~14 фото). Отбраковка артефактов и перегенерация до достижения кинематографичного качества через Hailuo AI.
2. **Post-Production (Монтаж):** Бесшовная склейка, цветокоррекция, музыка, звуковые эффекты и наложение титров в стилистике бренда.
3. **Рендер (Multi-Format):** Экспорт каждого из 42 роликов в **два формата**:
   - Desktop (Горизонтальный 16:9).
   - Mobile (Вертикальный/Квадратный 9:16 / 4:5 с кадрированием по центру фокуса).

### 💰 Финансовая оценка (ИИ Производство со скидкой 30%)
- **Стоимость 1 ролика в 2-х форматах (включая расходы на подписки Hailuo Pro):** **$84 – $126** *(вместо $120 – $180)*.
- **Итоговая стоимость 42 мультиформатных роликов:** **$3,500 – $5,250** *(вместо $5,000 – $7,500)*.
- **Сроки:** 4–6 недель.
