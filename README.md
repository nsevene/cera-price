# Оценка стоимости реализованного проекта (тема CERA + Плагины)

В этом документе представлена финальная оценка стоимости **уже разработанного функционала**, которую вы можете использовать для презентации заказчику или инвестору. Оценка базируется на рыночных ставках Middle+/Senior веб-разработчиков и Creative Frontend инженеров (специалистов по 3D, Canvas и React).

Общий объем работ по проекту оценивается примерно в **280 – 380 часов** высококвалифицированной разработки.

---

## Детализация сметы по компонентам

### 1. Фронтенд: WebGL / 3D Canvas / Интерактив (Плагин Cera 3D Catalog)
*Самая технически сложная и дорогая часть проекта.*
- Процедурная генерация текстур мрамора (Canvas API, алгоритмы шума).
- Кинематографичный многоступенчатый прелоадер (GSAP, SVG-анимации, чертежи, частицы).
- Уникальный физический движок скролла (инерция, параллакс, 3D-перспектива).
- Динамическая привязка фильтров WooCommerce без перезагрузки страниц.
- **Оценка:** ~60-80 часов
- **Стоимость:** **$2,500 – $4,000**

### 2. Frontend: ReactJS Слайдер (Плагин Cera Portfolio Elementor)
- Разработка сложного React-приложения, встроенного в среду WordPress.
- Настройка Webpack/Babel сборки.
- Реализация трех продвинутых режимов отображения (бесконечный Stream, горизонтальная Gallery, 3D Stack для мобильных).
- Связка с Elementor API для создания кастомных виджетов.
- **Оценка:** ~50-70 часов
- **Стоимость:** **$2,000 – $3,000**

### 3. Backend: Архитектура темы CERA и CERA Helper
- Разработка Custom Post Types (`portfolios`) и кастомных таксономий.
- Создание 21 уникального виджета для Elementor.
- Реализация сложной сквозной логики "пустышек" (отключение ссылок на карточках без проектов).
- Глобальные настройки темы (Customizer) и кастомная обработка типографики.
- **Оценка:** ~60-80 часов
- **Стоимость:** **$1,500 – $2,500**

### 4. Верстка и UI/UX Анимации (Тема CERA)
- Анимации доскролла (Ambient effects).
- Логика автопереходов с прогресс-баром и таймером прерывания.
- "Орбы" (плавающие шары с CSS-градиентами) на страницах портфолио.
- Дизайн и адаптация всех элементов интерфейса (Mobile, Tablet, Desktop).
- **Оценка:** ~40-60 часов
- **Стоимость:** **$1,000 – $1,800**

### 5. Интеграция WooCommerce (Премиум-кастомизация)
- Переопределение стандартного поведения WooCommerce для создания полноэкранных карточек товара ("чистый" дизайн без сайдбаров).
- Разработка безопасной AJAX-формы "Запросить образец".
- **Оценка:** ~30-40 часов
- **Стоимость:** **$800 – $1,200**
